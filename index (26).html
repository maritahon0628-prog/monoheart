API 設定

或手动输入：

CHARACTER PROFILE

角色每次自动回复的消息条数将在此范围内，根据角色性格和对话情境自动决定。

1.0

1.0 = 平衡（贴合人设+适当创意）。数值越小越保守贴合人设，数值越大回复越有创意生动。

关闭：角色依照聊天内容推断时间。开启：角色会感知手机系统时间，并据此调整回复逻辑。

开启後，角色可根據情況從表情包库中選擇表情包发送（頻率根據人設決定）。

开启後，角色可隨機混合语音信息回覆（可與文字、表情包混合发送）。

100

角色回覆時會記住最近的對話條數（包含雙方消息）。數值越大记忆越完整，但API消耗越多。最小10條，最大30000條，預設100條。

小時

开启後，角色會在設定的時間間隔內自動发送一條信息給你。最少每1小時，最多每24小時。

开启後，角色會在每天設定的時間自動生成一篇日記/朋友圈動態。
